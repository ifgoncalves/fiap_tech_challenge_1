# fiap_tech_challenge_1

Projeto de EDA com o objetivo de estudar o NPS de uma empresa, as variáveis que mais o impactam e possíveis recomendações para a melhoria desse indicador

* Objetivo do projeto: Explanar e propor melhorias que impactem diretamente na jornada do cliente, a partir da investigação do atual NPS da empresa;

* Descrição da base de dados: A base de dados contém 2.500 avaliações de clientes, com indicadores operacionais, como tempo de entrega, e informaçõpes básicas do cliente também, como idade e região;

* Metodologia utilizada: Foi realizada uma análise exploratória de dados para entender a relação das variáveis com o NPS, seguido da construção de um storytelling  para comunicar os achados;

* Como reproduzir os resultados: O notebook consome a base fonte e gera as visões e números necessários para a apresentação executiva!
